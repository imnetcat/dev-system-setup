# dev-system-setup
My dev system setup and appearance for arch-based OS

### Prerequirements
- Set up Arch-based OS with KDE Desktop
- Set up internet

### Install

```
sudo pacman -Syu git --noconfirm
cd ~
git clone https://github.com/imnetcat/dev-system-setup
cd dev-system-setup
chmod +x setup.sh
./setup.sh
```
